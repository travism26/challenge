##Getting started

###Input
1. The first input will be the size of the puzzle, Ie: 3 = 3x3 grid, 
2. The remaining inputs will be connections within the grid.
3. To finish just enter an empty value in other words just press enter. 

####Input example
3 <br/>
0,3 <br/>
3,4 <br/>
